# Restaurant-Menu
Developed a structured restaurant menu webpage using HTML. The project demonstrates the use of nested lists to organize menu items into categories such as Starters, and Desserts, including Veg and Non-Veg sections. It also includes special combo offers to represent real world menu design. This project highlights fundamental HTML concepts.
Features:
Nested list structure
Veg and Non-veg sections
Special combo menu
Technologies Used:
HTML
